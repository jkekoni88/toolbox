Yleistä

  • Natiivikoko kaikissa generaattoreissa on 1920×1080. Esikatselu (canvas) on skaalattu 960×540.
  • “Lataa kuva” tekee aina 1920×1080 PNG:n käyttäen samaa geometriaa kuin esikatselussa.

Miksi “Lataa kuva” voi joskus epäonnistua?

Selaimen suojausrajoitus (CORS): jos canvasille piirretty kuva on ladattu eri verkkotunnuksesta ilman sallivia CORS-otsikoita, selain estää kuvan tallentamisen (esim. toBlob()). Oireena näkyy usein SecurityError.

  • Ratkaisu 1: Pidä kaikki kuvat samalta originilta (sama protokolla, host ja portti).
  • Ratkaisu 2: Tarjoa kuvat CORS-otsikolla Access-Control-Allow-Origin ja lataa ne crossorigin="anonymous" -asetuksella.
  • Ratkaisu 3: Älä käytä file://-polkuja; aja paikallinen http-palvelin.

KehysGen

KehysGen lisää kuvan päälle valitun kehyksen ja tarjoaa perussäädöt (skaala, kirkkaus, kontrasti, kylläisyys, hue, vinjetti, film grain, blur, peilaus). Kuvaa voi raahata suoraan esikatselussa.

Kehysten lisääminen

  1. Tallenna 1920×1080 läpinäkyvät kehykset kansioon /frames.
  2. Luo/päivitä /frames/manifest.json: 
    {
  "frames": [
    { "id": "kehys_esimerkki", "name": "Esimerkki", "file": "kehys_esimerkki.png" },
    { "id": "kehys_tv", "name": "TV-kehys", "file": "kehys_tv.png" }
  ]
}

GridGen

GridGen tekee monisoluisia layoutteja (2 saraketta, 2 riviä, 2×2, 3 saraketta, kaaret/nuolet, timantti + 4, sekä Ympyrä + tausta).

PoliisiGen

PoliisiGen tekee sinisen vakio-taustan päälle keskitetyn sisältöalueen (maksimi leveys 1375 px), johon tiputetaan 1–3 kuvaa. Jokaisen solun ympärillä on 16 px valkoinen stroke. Ylä- ja alareunaan on erilliset väli-sliderit.

Tekstipalkkien lisääminen

  1. Tallenna PNG:t kansioon /poliisiframes:
    • kehys_kadonnut.png
    • kehys_poliisi_pyytaa_havaintoja.png
    • kehys_poliisi_pyytaa_vihjeita.png
    • kehys_soita_112.png
  2. Lisää /poliisiframes/manifest.json vastaavilla riveillä.

Vianetsintä

  • SecurityError toBlob(): tarjoa kuvat samalta originilta tai käytä CORS + crossorigin="anonymous".
  • Kehyslistat eivät päivity: varmista manifest.json -polut.
  • Esikatselu/vienti eri kohdassa: varmistu, että samat laskennat (offset/scale) ovat sekä preview- että export-koodissa.
